# sqflint
Provides basic syntax error checking for SQF language.

## Requirements
* JRE 1.8

## Development requirements
* JDK 1.8
* javacc 6.0
 - javacc is used to build parser
 - build process expects `javacc.bat` file in PATH, which executes javacc

* java-json library
* commons-cli library
